# Forever-Website

A full-stack MERN (MongoDB, Express, React, Node.js) e-commerce platform specialized in clothing retail, offering a seamless shopping experience with admin management capabilities.

## 🎯 Project Overview

Forever-Website is a modern e-commerce application built with the MERN stack, designed to provide users with an intuitive interface for browsing and purchasing clothing items. The platform includes customer-facing features, comprehensive admin controls, and secure payment processing.

## 🛠️ Tech Stack

### Frontend
- **React** - UI library for building interactive components
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first CSS framework for styling
- **Axios** - HTTP client for API requests
- **React Router** - Client-side routing

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB object modeling
- **JWT** - Authentication & authorization
- **Stripe/Razorpay** - Payment processing

### Admin Panel
- **React** - Admin dashboard interface
- **Redux** - State management
- **React Admin** - Admin dashboard template

## ✨ Features

### Customer Features
- 🛍️ Browse clothing products by categories
- 🔍 Search and filter products
- 🛒 Shopping cart functionality
- ❤️ Wishlist management
- 👤 User authentication & profile management

- 📝 Product reviews & ratings

### Admin Features
